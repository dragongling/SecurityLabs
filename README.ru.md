# SecurityLabs
Лабораторные работы для [Оренбургского Государственного Университета](http://osu.ru) по информационной безопасности
* MetalWorker.Cryptography - содержит вспомогательные классы
  * Различные методы для проверки чисел на простоту
  * Генератор случайных BigInteger
  * Генератор случайных простых BigInteger
  * Реализация блочного RSA
  * StringRSA - обёртка блочного RSA для строк
* PrimalityTests - программа для проверки чисел на простоту различными методами
* BlockRSA_GUI - GUI клиент для шифрования строк блочным RSA
* Digital Signature - GUI приложение для цифровой подписи файлов с помощью RSACryptoServiceProvider
* AccessControlMatrix - CLI симулятор матричной модели доступа пользователей к файлам
* DES - Форк [проекта на cyberforum от VV_RIP](http://www.cyberforum.ru/csharp-net/thread1120037.html), я пересобрал для VS2017 и добавил GUI
